#### 🗓️ 칭찬 일력  
- 매일 새로운 칭찬을 만나고 기록해요.

#### ✍️ 필사  
- 오늘의 칭찬을 직접 따라 쓰며 마음에 새겨요.

#### 🤝 칭구 만나기  
- 나와 잘 맞는 성격의 칭구를 선택하고 따뜻한 대화를 나눠봐요.

#### 💌 칭찬 카드  
- 마음에 남는 칭찬은 카드로 저장하고 공유해요.

#### 📚 아카이브  
- 저장된 칭찬과 카드를 언제든 다시 보기.
